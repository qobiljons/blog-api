Blog API Documentation

Welcome to the Blog API documentation. This API allows you to manage blog posts, comments, and user accounts programmatically.

Base URL

The base URL for all API requests is:

https://yourapi.com/api/

Authentication

All endpoints require authentication via a token. You must include the token in the Authorization header of your requests.

Authorization: Bearer <your-token>

Endpoints

User Endpoints

Register a New User

Endpoint: POST /users/register/
Request Body: 
{
    "username": "string",
    "password": "string",
    "email": "string"
}
Response: 
{
    "id": "integer",
    "username": "string",
    "email": "string"
}

Login

Endpoint: POST /users/login/
Request Body: 
{
    "username": "string",
    "password": "string"
}
Response: 
{
    "token": "string"
}
