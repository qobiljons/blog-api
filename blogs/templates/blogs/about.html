Get Started with Blog API

Welcome! This guide will help you get started with the Blog API, allowing you to manage blog posts, comments, and user accounts programmatically.

Step 1: Base URL

All API requests are made to the following base URL:

https://yourapi.com/api/

Step 2: Authentication

All endpoints require authentication via a token. You must include the token in the Authorization header of your requests.

Authorization: Bearer <your-token>

Step 3: Register a New User

Register a new user by sending a POST request to the following endpoint:

POST /users/register/

Request Body:

{
    "username": "string",
    "password": "string",
    "email": "string"
}

Response:

{
    "id": "integer",
    "username": "string",
    "email": "string"
}

Step 4: Login

Login by sending a POST request to the following endpoint:

POST /users/login/

Request Body:

{
    "username": "string",
    "password": "string"
}

Response:

{
    "token": "string"
}

Step 5: Using the Token

Include the token in the Authorization header of all subsequent requests:

curl -H "Authorization: Bearer <your-token>" https://yourapi.com/api/posts/
